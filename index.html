Start from Scratch: Build an App, Containerize it, Deploy with Kubernetes

Steps for building an app to run with containers and Kubernetes:

1. Install any code editor of your choice (I used VS Code).

2. Install Python for your machine.

3. Optional: Connect Python manually to your path.

Run this in terminal: open -e ~/.zprofile  
Paste this in the file that opens:  
export PATH="/Library/Frameworks/Python.framework/Versions/3.13/bin:$PATH"  
Then run this: source ~/.zprofile  
Now run: python --version

4. Create a simple helloworld.py Python script to test it prints to terminal.

5. Install Flask to help with the API:

pip install Flask

7. Run the app:

python3 app.py

8. Open your browser and go to: http://localhost:3001/app

You can update the route or port number in your code if needed.

Now it's time to start working with containers:

1. Install Podman on your desktop (download it from the official site).

2. Open your terminal and run podman and a list of commands should pop up:

podman

3. Create a requirements.txt file and add dependencies (example: flask)

5. Run the container:

podman machine init  
podman machine start  
podman build -t <your_folder_name> .

If you get a pull limit error, try again or create a Docker Hub account.

If you get a COPY error, make sure you're referencing the right folders/files in your Dockerfile.

6. Run this to check if your image built successfully:

podman images

7. Start your container:

podman run -it -p 3001:3000 --name anjali2 anjali_hw

You should now see the local host pop up and your app running inside the container!
